jQuery Form Styler <sup>[1.2.1](http://dimox.name/jquery-form-styler/#log)</sup>
==================

jQuery-плагин для стилизации элементов html-форм:

* `<input type="checkbox">`
* `<input type="radio">`
* `<input type="file">`
* `<select>`
